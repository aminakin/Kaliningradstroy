Вышки передвижные.

Вышки передвижные предназначены для выполнения отделочных, монтажных и ремонтных работ, как снаружи, так и внутри помещений, в качестве средств подмащивания.

За основу конструкции принята Вышка Главмосстроя, представляющая собой лёгкую каркасную пространственную систему, состояющую из отдельных взаимо-заменяемых плоских секций (рамок), которые собираются в шахматном порядке на опрной тележке.

Для изготовления элементов Вышки применяются стальные водогазопроводные трубы, а для настилов доски обрезные хвойных пород.

По размерам в плане Вышки изготавливаются 3-х типоразмеров: 2,0х2,0; 1,2х1,2. Соответственно для сборки всех типоразмеров Вышек используется два вида рамок, с помощью которых Вышки трансформируются в любой из указанных типоразмеров.

Во избежание качки во время производства работ основание Вышки оснащено винтовыми опорами.
